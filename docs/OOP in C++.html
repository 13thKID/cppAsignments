physics course database
  • Please note: expected coding time ~5 hours over two weeks.
  • Separate goal for first week, which will be further developped in second week.
  • Write a C++ program to store and print physics courses, plus some statistical information
  • Allow the user to enter an arbitrary number of courses from a file
  • Ask the user to provide an course mark (as a float/double), course code (as an integer, e.g. 30762) and title (as a string, e.g. Object-Oriented Programming in C++), all on a single line

Assignment 2
physics course database, second week
  • Extension
  • Add the following functionality to the code (you now have to use vectors for storage).
  • Be efficient in storage
  • Use a string-stream to create a string containing the full course title, e.g.
    PHYS 30762 Object-Oriented Programming in C++
  • Each of these strings should be stored in a single vector
  • Print out the full course list using an iterator
  • Your code should also be able to print out a list of courses for a particular year, as identified by the first digit of the course code, including mean and standard deviation for those courses.
